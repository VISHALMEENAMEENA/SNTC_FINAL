## ğŸ”§ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd sntc-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
